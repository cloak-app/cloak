## 安装

前往 [Release page](https://github.com/Xav1erSue/cloak/releases) 下载对应平台的安装包。

## 功能

### 已实现功能

- [x] 导入 `TXT` 小说文件。
- [x] 章节解析
- [x] 章节搜索/章节跳转
- [x] 自动保存阅读进度
- [x] 自定义字体
- [x] 自定义字体大小
- [x] 自定义字体粗细
- [x] 自定义行间距
- [x] 自定义字符间距
- [x] 自定义字符间距
- [x] 自定义翻页快捷键
- [x] 自定义老板键
- [x] 自动检查更新/自动安装更新

### 待实现功能（开发中）

- [ ] 导入 `EPUB` 小说文件
- [ ] 支持自定义章节解析正则表达式
- [ ] i18n(目前仅支持中文)

### 支持平台

- [x] Windows
- [x] Mac
- [x] Linux
